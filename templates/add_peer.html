Add a New Peer
















Instructions

Generating Keys:

Run the following commands to generate a private and public key:

wg genkey | tee privatekey | wg pubkey > publickey

Field Descriptions:

  • Name: A descriptive name for the peer.
  • Public Key: The public key generated for the peer.
  • Allowed IPs: The IPs allowed for this peer (e.g., 10.0.0.2/32).
  • Endpoint: (Optional) The peer's endpoint address.
  • Persistent Keepalive: (Optional) Interval to keep the connection alive.
